# Prévisualisation client

La prévisualisation client vous permet de voir exactement ce que voit le client dans son interface.

## Activer la prévisualisation

1. Accédez à la liste des produits de votre projet
2. Cliquez sur **Prévisualisation client** dans la barre d'outils
3. La vue passe en mode prévisualisation

## Ce que vous voyez en prévisualisation

En mode prévisualisation, vous verrez uniquement :

### Produits visibles
- Les produits marqués comme "visibles pour le client"
- Ou tous les produits si le paramètre de partage est configuré ainsi

### Informations affichées
- Nom du produit
- Description (si partagée)
- Images (si partagées)
- Prix (selon les paramètres de partage des prix)
- Quantité (si visible)

### Informations masquées
- Prix internes (si masqués)
- Notes internes
- Fournisseurs (si non partagés)
- Références internes

## Utiliser la prévisualisation

### Vérifier la visibilité des produits
1. Activez la prévisualisation
2. Parcourez la liste des produits
3. Vérifiez que seuls les produits souhaités sont visibles

### Tester les filtres
1. Activez la prévisualisation
2. Utilisez les filtres comme le client le ferait
3. Vérifiez que les résultats correspondent à vos attentes

### Vérifier l'affichage des prix
1. Activez la prévisualisation
2. Vérifiez que les prix affichés correspondent aux paramètres de partage
3. Si les prix sont masqués, confirmez qu'ils n'apparaissent pas

## Désactiver la prévisualisation

Pour revenir à la vue normale :

1. Cliquez sur **Quitter la prévisualisation** dans la barre d'outils
2. Ou cliquez sur l'icône de prévisualisation à nouveau

## Paramètres de partage

La prévisualisation respecte tous les paramètres de partage configurés dans les **Paramètres** du projet :

- **Partage des produits** : Quels produits sont visibles
- **Partage des prix** : Si les prix sont affichés
- **Partage des fichiers** : Quels fichiers sont accessibles

Pour modifier ces paramètres, quittez la prévisualisation et accédez aux **Paramètres** du projet.

## Conseils

- Utilisez régulièrement la prévisualisation avant de partager un projet avec un client
- Vérifiez que les informations sensibles ne sont pas visibles
- Testez les différents filtres pour vous assurer que le client peut trouver les produits facilement
