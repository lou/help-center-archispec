# Installation de l'extension Chrome

L'extension Archispec permet d'extraire rapidement des produits depuis n'importe quel site web et de les ajouter directement à vos projets.

## Télécharger l'extension

L'extension Archispec est disponible sur le Chrome Web Store :

1. Accédez à la [page de l'extension](https://chromewebstore.google.com/detail/archispec/adokjifkncmndmiplmgdbofnpckkojpc)
2. Cliquez sur **Ajouter à Chrome**
3. Confirmez l'installation en cliquant sur **Ajouter l'extension**

## Première utilisation

Après l'installation :

1. L'icône Archispec apparaît dans la barre d'outils de Chrome
2. Cliquez sur l'icône pour ouvrir l'extension
3. Connectez-vous avec vos identifiants Archispec
4. L'extension est maintenant prête à être utilisée !

## Vérifier l'installation

Pour vérifier que l'extension est bien installée :

1. Cliquez sur l'icône de l'extension dans la barre d'outils
2. Vous devriez voir l'interface de connexion ou votre espace de travail
3. Si vous voyez l'interface, l'installation est réussie !

## Désinstaller l'extension

Si vous souhaitez désinstaller l'extension :

1. Faites un clic droit sur l'icône de l'extension
2. Sélectionnez **Gérer l'extension**
3. Cliquez sur **Supprimer**

Ou :

1. Allez dans `chrome://extensions`
2. Trouvez l'extension Archispec
3. Cliquez sur **Supprimer**
