# Installation de l'extension

L'extension Archispec permet d'extraire rapidement des produits depuis n'importe quel site web et de les ajouter à vos projets.

## Navigateurs supportés

L'extension est disponible pour :
- **Google Chrome** : Version recommandée
- **Microsoft Edge** : Basé sur Chromium
- **Mozilla Firefox** : Disponible prochainement

## Installation depuis le Chrome Web Store

### Méthode recommandée

1. Ouvrez Google Chrome
2. Accédez au [Chrome Web Store](https://chrome.google.com/webstore)
3. Recherchez "Archispec" dans la barre de recherche
4. Cliquez sur **Ajouter à Chrome**
5. Confirmez l'installation en cliquant sur **Ajouter l'extension**

L'extension est maintenant installée et l'icône Archispec apparaît dans la barre d'outils de Chrome.

## Installation manuelle (mode développeur)

Si l'extension n'est pas encore disponible sur le Chrome Web Store :

1. Téléchargez le fichier ZIP de l'extension
2. Décompressez le fichier dans un dossier
3. Ouvrez Chrome et accédez à `chrome://extensions/`
4. Activez le **Mode développeur** (en haut à droite)
5. Cliquez sur **Charger l'extension non empaquetée**
6. Sélectionnez le dossier décompressé
7. L'extension est maintenant installée

## Première configuration

### Connexion

1. Cliquez sur l'icône de l'extension dans la barre d'outils
2. Cliquez sur **Se connecter**
3. Entrez vos identifiants Archispec
4. Cliquez sur **Connexion**

L'extension est maintenant connectée à votre compte.

### Autorisations

L'extension demande les autorisations suivantes :
- **Accès aux sites web** : Pour extraire les informations des produits
- **Stockage** : Pour sauvegarder vos préférences
- **Notifications** : Pour vous informer des actions réussies

Acceptez ces autorisations pour que l'extension fonctionne correctement.

## Vérifier l'installation

Pour vérifier que l'extension est correctement installée :

1. Cliquez sur l'icône de l'extension
2. Vous devriez voir l'interface de connexion ou votre tableau de bord
3. Si vous voyez une erreur, consultez la section [Dépannage](#dépannage)

## Mettre à jour l'extension

### Mise à jour automatique

Si l'extension est installée depuis le Chrome Web Store, elle se met à jour automatiquement.

### Mise à jour manuelle

1. Accédez à `chrome://extensions/`
2. Cliquez sur **Mettre à jour** si une mise à jour est disponible
3. Ou désinstallez et réinstallez l'extension depuis le Chrome Web Store

## Désinstaller l'extension

1. Accédez à `chrome://extensions/`
2. Trouvez l'extension Archispec
3. Cliquez sur **Supprimer**
4. Confirmez la suppression

## Dépannage

### L'icône n'apparaît pas

1. Vérifiez que l'extension est activée dans `chrome://extensions/`
2. Si l'icône est masquée, cliquez sur l'icône de puzzle dans la barre d'outils
3. Épinglez l'extension Archispec pour qu'elle reste visible

### L'extension ne se connecte pas

1. Vérifiez votre connexion internet
2. Vérifiez que vos identifiants sont corrects
3. Essayez de vous déconnecter et de vous reconnecter
4. Si le problème persiste, contactez le support

### L'extension ne détecte pas les produits

1. Assurez-vous d'être sur une page de produit (pas sur une page d'accueil)
2. Actualisez la page
3. Cliquez à nouveau sur l'icône de l'extension
4. Certains sites web peuvent ne pas être compatibles
