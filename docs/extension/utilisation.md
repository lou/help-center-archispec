# Utilisation de l'extension

Apprenez à utiliser l'extension Archispec pour extraire et ajouter des produits rapidement.

## Extraire un produit

### Sur une page de produit

1. Naviguez sur un site web contenant un produit (ex: site de fournisseur, e-commerce)
2. Assurez-vous d'être sur la page de détail du produit
3. Cliquez sur l'icône de l'extension Archispec dans la barre d'outils
4. L'extension analyse automatiquement la page et extrait les informations :
   - Nom du produit
   - Prix
   - Description
   - Images
   - Référence (si disponible)

### Vérifier les informations extraites

1. Après le clic sur l'extension, une fenêtre s'ouvre
2. Vérifiez les informations extraites :
   - **Nom** : Vérifiez que le nom est correct
   - **Prix** : Vérifiez le prix extrait
   - **Images** : Vérifiez que les bonnes images sont sélectionnées
   - **Description** : Vérifiez ou complétez la description

### Modifier les informations

Vous pouvez modifier toutes les informations avant d'ajouter le produit :

1. Cliquez sur un champ pour le modifier
2. Entrez les nouvelles informations
3. Les modifications sont sauvegardées automatiquement

## Ajouter un produit à un projet

### Sélectionner le projet

1. Dans la fenêtre de l'extension, cliquez sur **Sélectionner un projet**
2. Recherchez le projet dans la liste
3. Sélectionnez le projet de destination

### Ajouter au projet

1. Vérifiez que toutes les informations sont correctes
2. Cliquez sur **Ajouter au projet**
3. Une confirmation apparaît
4. Le produit est ajouté au projet avec toutes ses informations

### Ajouter au catalogue

Au lieu d'ajouter directement à un projet, vous pouvez ajouter au catalogue :

1. Dans la fenêtre de l'extension, cliquez sur **Ajouter au catalogue**
2. Le produit est ajouté à votre bibliothèque
3. Vous pourrez l'ajouter à un projet plus tard depuis le catalogue

## Fonctionnalités avancées

### Extraire plusieurs produits

Sur une page listant plusieurs produits :

1. Cliquez sur l'icône de l'extension
2. L'extension détecte automatiquement tous les produits de la page
3. Sélectionnez les produits à ajouter
4. Cliquez sur **Ajouter la sélection**

### Modifier les images

1. Dans la fenêtre de l'extension, cliquez sur une image
2. Vous pouvez :
   - **Supprimer** : Retirer une image
   - **Réorganiser** : Glisser-déposer pour changer l'ordre
   - **Ajouter** : Ajouter des images depuis votre ordinateur

### Ajouter des notes

1. Dans la fenêtre de l'extension, trouvez le champ **Notes**
2. Ajoutez des notes personnelles sur le produit
3. Les notes sont incluses avec le produit

## Raccourcis clavier

- **Ctrl/Cmd + Shift + A** : Ouvrir l'extension sur la page actuelle
- **Échap** : Fermer la fenêtre de l'extension

## Paramètres de l'extension

### Accéder aux paramètres

1. Cliquez sur l'icône de l'extension
2. Cliquez sur l'icône d'engrenage (⚙️)
3. Les paramètres s'ouvrent

### Paramètres disponibles

#### Projet par défaut
Définissez un projet par défaut pour ajouter rapidement des produits sans sélectionner le projet à chaque fois.

#### Extraction automatique
Activez l'extraction automatique des informations pour accélérer le processus.

#### Notifications
Activez ou désactivez les notifications de confirmation lors de l'ajout de produits.

#### Qualité des images
Choisissez la qualité des images à extraire :
- **Haute qualité** : Images en haute résolution (plus lourd)
- **Qualité moyenne** : Bon compromis
- **Basse qualité** : Images légères (plus rapide)

## Conseils d'utilisation

### Meilleures pratiques

1. **Vérifiez toujours les informations** : L'extraction automatique peut parfois faire des erreurs
2. **Complétez les descriptions** : Ajoutez des détails manquants pour faciliter la recherche plus tard
3. **Organisez par projet** : Ajoutez directement les produits au bon projet
4. **Utilisez le catalogue** : Pour les produits récurrents, ajoutez-les d'abord au catalogue

### Sites web compatibles

L'extension fonctionne mieux sur :
- Sites e-commerce standards (WooCommerce, Shopify, etc.)
- Sites de fournisseurs avec structure HTML standard
- Pages avec métadonnées structurées (schema.org)

Certains sites peuvent nécessiter une extraction manuelle des informations.

## Dépannage

### L'extension ne détecte pas le produit

1. Assurez-vous d'être sur une page de produit (pas une liste)
2. Actualisez la page et réessayez
3. Essayez de cliquer manuellement sur les éléments de la page
4. Certains sites peuvent ne pas être compatibles

### Les images ne s'extraient pas

1. Vérifiez que les images sont bien chargées sur la page
2. Essayez d'actualiser la page
3. Vous pouvez toujours ajouter des images manuellement

### Le prix est incorrect

1. Vérifiez le prix sur la page web
2. Modifiez manuellement le prix dans la fenêtre de l'extension
3. Certains sites affichent les prix de manière non standard
