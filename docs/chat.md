# Chat

Communiquez avec votre équipe et vos clients directement dans le projet.

## Accéder au chat

1. Ouvrez votre projet
2. Cliquez sur **Chat** dans le menu latéral du projet
3. La fenêtre de chat s'ouvre

## Envoyer un message

1. Tapez votre message dans la zone de texte en bas
2. Cliquez sur **Envoyer** ou appuyez sur **Entrée**

## Types de messages

### Message texte
Un simple message texte pour communiquer avec l'équipe.

### Mentionner un membre
Pour attirer l'attention d'un membre spécifique :

1. Tapez **@** suivi du nom du membre
2. Sélectionnez le membre dans la liste déroulante
3. Le membre recevra une notification

### Joindre un fichier
1. Cliquez sur l'icône de pièce jointe
2. Sélectionnez le fichier à joindre
3. Le fichier est téléversé et ajouté au message

### Répondre à un message
1. Survolez le message auquel vous souhaitez répondre
2. Cliquez sur **Répondre**
3. Tapez votre réponse
4. Votre réponse sera liée au message original

## Notifications

### Notifications en temps réel
Vous recevez des notifications lorsque :
- Quelqu'un vous mentionne (@votre_nom)
- Quelqu'un répond à votre message
- Un nouveau message est posté dans le chat

### Paramètres de notifications
Configurez vos préférences de notifications :

1. Cliquez sur les paramètres du chat (icône d'engrenage)
2. Configurez :
   - **Notifications sonores** : Activer/désactiver les sons
   - **Notifications push** : Recevoir des notifications même quand l'application est fermée
   - **Notifications par email** : Recevoir un résumé par email

## Historique des messages

Tous les messages sont conservés dans l'historique du chat. Vous pouvez :
- Faire défiler vers le haut pour voir les messages précédents
- Rechercher dans l'historique avec la barre de recherche
- Filtrer par membre ou par date

## Épingler un message

Pour garder un message important visible :

1. Survolez le message
2. Cliquez sur **Épingler**
3. Le message apparaît en haut du chat

Pour désépingler :
1. Cliquez sur le message épinglé
2. Cliquez sur **Désépingler**

## Supprimer un message

1. Survolez votre message
2. Cliquez sur le menu (⋮)
3. Sélectionnez **Supprimer**
4. Confirmez l'action

**Note** : Vous ne pouvez supprimer que vos propres messages.

## Chat avec le client

Si un client a été invité au projet, il peut également utiliser le chat :

- Le client voit tous les messages du chat
- Le client peut envoyer des messages
- Le client reçoit des notifications pour les mentions

Pour désactiver le chat pour le client, modifiez les paramètres de partage dans les **Paramètres** du projet.

## Raccourcis clavier

- **Entrée** : Envoyer le message
- **Shift + Entrée** : Aller à la ligne
- **Ctrl/Cmd + K** : Ouvrir la recherche dans l'historique
- **Ctrl/Cmd + F** : Rechercher dans les messages

## Conseils

- Utilisez les mentions (@nom) pour attirer l'attention sur des messages importants
- Épinglez les messages contenant des informations importantes
- Joignez des fichiers directement dans le chat pour un partage rapide
- Consultez régulièrement l'historique pour ne rien manquer
