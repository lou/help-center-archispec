# Archiver un projet

L'archivage permet de conserver un projet tout en le retirant de votre liste de projets actifs.

## Archiver un projet

1. Accédez aux **Paramètres** du projet
2. Faites défiler jusqu'à la section **Actions**
3. Cliquez sur **Archiver le projet**
4. Confirmez l'action dans la boîte de dialogue

Le projet sera immédiatement archivé et retiré de votre liste de projets actifs.

## Restaurer un projet archivé

1. Accédez à la [liste des projets](https://archispec.fr/app/projects)
2. Cliquez sur le projet que vous souhaitez restaurer
4. Accédez aux **Paramètres** du projet
5. Cliquez sur **Restaurer le projet**

Le projet redeviendra actif et réapparaîtra dans votre liste de projets.

## Effets de l'archivage

Lorsqu'un projet est archivé :
- ✅ Tous les produits et fichiers sont conservés
- ✅ L'historique du chat est conservé
- ✅ Les clients peuvent toujours accéder au projet
- ❌ Le projet n'apparaît plus dans la liste des projets actifs

## Restaurer un projet archivé

Pour restaurer un projet archivé :

1. Accédez aux **Paramètres** du projet archivé
2. Cliquez sur **Archivage**
3. Cliquez sur **Restaurer le projet**
4. Le projet redeviendra actif et réapparaîtra dans votre liste de projets
