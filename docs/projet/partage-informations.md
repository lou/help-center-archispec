# Partage des informations

Contrôlez quelles informations sont partagées avec les clients.

## Paramètres de partage

1. Accédez aux **Paramètres** du projet
2. Cliquez sur **Partage** dans le menu latéral
3. Configurez les options de partage

## Options de partage disponibles

### Partager les prix avec le client

Activez cette option pour que les clients puissent voir les prix des produits dans leur interface.

- **Activé** : Les clients voient les prix des produits
- **Désactivé** : Les prix sont masqués pour les clients

### Partager les liens avec le client

Activez cette option pour que les clients puissent voir les liens sources (URLs) des produits.

- **Activé** : Les clients voient les liens vers les sites web des produits
- **Désactivé** : Les liens sont masqués pour les clients
