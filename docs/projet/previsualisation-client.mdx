# Prévisualisation client

La prévisualisation client vous permet de visualiser exactement l’interface telle qu’elle apparaît pour le client.
En mode prévisualisation, toutes les actions sont désactivées, ce qui élimine tout risque de manipulation ou d’action involontaire.

## Activer/désactiver la prévisualisation client

1. Accédez à la page de votre projet
2. Cliquez sur l'**icône oeil** à droite du nom du projet
3. Vous êtes maintenant en mode prévisualisation, une **bannière orange** en haut de la page vous le confirme
4. Pour sortir du mode prévisualisation, cliquez sur la **croix** à droite de la bannière orange
