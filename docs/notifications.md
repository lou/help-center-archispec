# Notifications

Gérez vos notifications pour rester informé des activités importantes dans vos projets.

## Types de notifications

### Notifications de projet
Vous recevez des notifications lorsque :
- Un nouveau membre est ajouté au projet
- Un client est invité ou accepte une invitation
- Un produit est ajouté ou modifié
- Un fichier est partagé
- Un message vous est adressé dans le chat

### Notifications de mention
- Quand quelqu'un vous mentionne dans le chat (@votre_nom)
- Quand quelqu'un répond à votre message

### Notifications d'entreprise
- Quand un nouveau membre rejoint l'entreprise
- Quand vos permissions changent
- Quand des paramètres d'entreprise sont modifiés

## Accéder aux notifications

### Centre de notifications
1. Cliquez sur l'icône de cloche dans la barre de navigation
2. Toutes vos notifications s'affichent
3. Les notifications non lues sont marquées d'un indicateur

### Notifications par email
Selon vos paramètres, vous pouvez recevoir des notifications par email :
- Résumé quotidien
- Résumé hebdomadaire
- Notifications en temps réel

## Gérer les notifications

### Marquer comme lu
1. Cliquez sur une notification
2. Elle est automatiquement marquée comme lue

Ou :

1. Cliquez sur **Tout marquer comme lu** dans le centre de notifications
2. Toutes les notifications sont marquées comme lues

### Supprimer une notification
1. Survolez la notification
2. Cliquez sur l'icône de suppression (×)
3. La notification est supprimée

### Supprimer toutes les notifications
1. Dans le centre de notifications
2. Cliquez sur **Tout supprimer**
3. Confirmez l'action

## Paramètres de notifications

Configurez vos préférences de notifications :

1. Cliquez sur votre profil (en haut à droite)
2. Sélectionnez **Paramètres**
3. Cliquez sur **Notifications**

### Notifications en temps réel
Activez ou désactivez les notifications en temps réel dans l'application.

### Notifications push
Recevez des notifications même quand l'application est fermée :
1. Activez les **Notifications push**
2. Autorisez les notifications dans votre navigateur

### Notifications par email
Configurez la fréquence des emails :
- **Jamais** : Aucun email
- **En temps réel** : Un email pour chaque notification
- **Résumé quotidien** : Un email par jour avec toutes les notifications
- **Résumé hebdomadaire** : Un email par semaine

### Notifications par type
Activez ou désactivez les notifications par type :
- **Projets** : Notifications liées aux projets
- **Chat** : Notifications de messages et mentions
- **Fichiers** : Notifications de partage de fichiers
- **Entreprise** : Notifications d'entreprise

### Notifications par projet
Pour un projet spécifique :

1. Ouvrez le projet
2. Accédez aux **Paramètres** du projet
3. Cliquez sur **Notifications**
4. Configurez les notifications pour ce projet :
   - **Toutes les notifications** : Recevoir toutes les notifications
   - **Notifications importantes uniquement** : Seulement les mentions et actions importantes
   - **Aucune notification** : Ne pas recevoir de notifications pour ce projet

## Raccourcis clavier

- **N** : Ouvrir/fermer le centre de notifications
- **Échap** : Fermer le centre de notifications

## Conseils

- Configurez vos préférences pour éviter d'être submergé de notifications
- Utilisez les résumés par email si vous préférez consulter les notifications en une fois
- Marquez les notifications comme lues régulièrement pour garder votre centre de notifications organisé
- Activez les notifications push pour les projets importants
