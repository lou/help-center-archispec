# Interface client

Guide pour utiliser l'interface client d'Archispec et consulter vos projets.

## Connexion

1. Accédez à l'interface client Archispec
2. Cliquez sur **Se connecter**
3. Entrez votre email et votre mot de passe
4. Cliquez sur **Connexion**

Si vous avez reçu une invitation par email, cliquez sur le lien dans l'email pour accéder directement au projet.

## Vue d'ensemble

L'interface client vous permet de :
- Consulter les projets partagés avec vous
- Voir les produits sélectionnés pour votre projet
- Télécharger les fichiers partagés
- Communiquer avec l'équipe via le chat
- Prévisualiser les devis et factures

## Liste des projets

### Accéder à vos projets

1. Dans le menu principal, cliquez sur **Projets**
2. Tous les projets partagés avec vous s'affichent

### Informations affichées

Pour chaque projet, vous voyez :
- **Nom du projet** : Nom du projet architectural
- **Architecte** : Nom de l'entreprise ou de l'architecte
- **Date de dernière mise à jour** : Quand le projet a été modifié pour la dernière fois
- **Statut** : État actuel du projet

### Ouvrir un projet

Cliquez sur un projet pour l'ouvrir et voir les détails.

## Produits

### Consulter les produits

1. Ouvrez un projet
2. Cliquez sur **Produits** dans le menu latéral
3. Tous les produits partagés avec vous s'affichent

### Informations des produits

Pour chaque produit, vous pouvez voir :
- **Nom** : Nom du produit
- **Description** : Description détaillée
- **Images** : Photos du produit
- **Prix** : Prix (si partagé par l'architecte)
- **Quantité** : Quantité dans le projet

### Filtrer les produits

Utilisez les filtres pour trouver des produits :
- **Par zone** : Filtrez par pièce ou zone
- **Par type** : Filtrez par type de produit
- **Recherche** : Recherchez par nom ou description

## Fichiers

### Accéder aux fichiers

1. Dans un projet, cliquez sur **Fichiers**
2. Tous les fichiers partagés avec vous s'affichent

### Télécharger un fichier

1. Cliquez sur le fichier
2. Le fichier s'ouvre dans un nouvel onglet
3. Utilisez le menu de votre navigateur pour télécharger

Ou :

1. Cliquez sur le menu (⋮) à côté du fichier
2. Sélectionnez **Télécharger**

### Types de fichiers

Vous pouvez consulter :
- **Plans** : Plans architecturaux (PDF, images)
- **Documents** : Devis, factures, contrats
- **Images** : Photos de référence, rendus

## Chat

### Utiliser le chat

1. Dans un projet, cliquez sur **Chat**
2. La fenêtre de chat s'ouvre
3. Tapez votre message et appuyez sur **Envoyer**

### Types de messages

- **Messages texte** : Communiquez avec l'équipe
- **Fichiers joints** : L'équipe peut partager des fichiers dans le chat
- **Mentions** : Si l'équipe vous mentionne (@votre_nom), vous recevrez une notification

### Historique

Tous les messages sont conservés. Vous pouvez faire défiler vers le haut pour voir l'historique complet.

## Notifications

### Recevoir des notifications

Vous recevez des notifications lorsque :
- Un nouveau produit est ajouté au projet
- Un fichier est partagé avec vous
- Quelqu'un vous mentionne dans le chat
- Le projet est mis à jour

### Consulter les notifications

1. Cliquez sur l'icône de cloche dans la barre de navigation
2. Toutes vos notifications s'affichent
3. Cliquez sur une notification pour accéder au contenu

## Exports

### Télécharger un devis

1. Dans un projet, cliquez sur **Exporter**
2. Sélectionnez **Devis PDF**
3. Le devis est généré et téléchargé

### Télécharger un cahier des charges

1. Dans un projet, cliquez sur **Exporter**
2. Sélectionnez **Cahier des charges**
3. Un PDF avec tous les produits et images est généré

## Paramètres du compte

### Accéder aux paramètres

1. Cliquez sur votre profil (en haut à droite)
2. Sélectionnez **Paramètres**

### Modifier votre profil

1. Dans les paramètres, cliquez sur **Profil**
2. Modifiez vos informations :
   - Nom et prénom
   - Email
   - Téléphone
   - Mot de passe
3. Cliquez sur **Enregistrer**

### Notifications

Configurez vos préférences de notifications :
- **Notifications par email** : Recevoir des emails pour les mises à jour importantes
- **Notifications push** : Recevoir des notifications dans le navigateur

## Conseils

- Consultez régulièrement vos projets pour voir les mises à jour
- Utilisez le chat pour poser des questions à l'équipe
- Téléchargez les fichiers importants pour les conserver
- Activez les notifications pour ne rien manquer
