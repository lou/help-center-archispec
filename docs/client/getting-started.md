# Bienvenue dans l'espace client

L'espace client Archispec vous permet de suivre vos projets architecturaux en temps réel et de communiquer directement avec votre architecte.

## Accéder à votre espace

1. Vous recevrez une invitation par email de la part de votre architecte
2. Cliquez sur le lien dans l'email
3. Créez votre compte ou connectez-vous si vous en avez déjà un
4. Acceptez l'invitation au projet

## Navigation

Une fois connecté, vous accédez à votre espace client avec :

- **Projets** : Liste de tous les projets partagés avec vous
- **Compte** : Vos informations personnelles et paramètres

## Voir vos projets

1. Dans le menu, cliquez sur **Projets**
2. Vous verrez tous les projets que votre architecte a partagés avec vous
3. Cliquez sur un projet pour voir les détails

## Comprendre un projet

Chaque projet contient :

- **Produits** : Liste de tous les produits sélectionnés pour votre projet
- **Chat** : Communication directe avec l'équipe projet
- **Fichiers** : Documents et plans partagés
- **Prix** : Vue d'ensemble des prix si activée par l'architecte
