# Consulter les projets

En tant que client, vous pouvez consulter les projets que votre architecte a partagés avec vous.

## Vue d'ensemble d'un projet

Lorsque vous ouvrez un projet, vous accédez à plusieurs sections :

### Produits
La liste complète des produits sélectionnés pour votre projet. Pour chaque produit, vous pouvez voir :
- Le nom et la référence
- Les images
- La description
- La quantité
- Le prix (si activé par l'architecte)
- La zone (pièce) où le produit sera utilisé

### Chat
Communiquez directement avec l'équipe projet pour poser des questions ou faire des commentaires.

### Fichiers
Accédez aux documents et plans partagés par l'architecte.

### Prix
Si l'architecte a activé l'affichage des prix, vous pouvez voir :
- Le prix unitaire de chaque produit
- Le prix total par produit
- Le prix total du projet

## Filtrer et rechercher

Dans la section **Produits**, vous pouvez :

- **Rechercher** : Utilisez la barre de recherche pour trouver un produit spécifique
- **Filtrer par zone** : Affichez uniquement les produits d'une zone particulière
- **Filtrer par type** : Filtrez par type de produit

## Voir les détails d'un produit

1. Cliquez sur un produit dans la liste
2. Vous verrez :
   - Toutes les images du produit
   - La description complète
   - Les informations techniques
   - Les commentaires et discussions liés au produit

## Communiquer avec l'équipe

### Chat du projet

1. Accédez à la section **Chat** du projet
2. Tapez votre message
3. Envoyez-le pour communiquer avec l'équipe

### Chat d'un produit

1. Ouvrez un produit
2. Cliquez sur **Chat**
3. Posez vos questions spécifiques sur ce produit

## Télécharger des fichiers

1. Accédez à la section **Fichiers** du projet
2. Cliquez sur un fichier pour le télécharger
3. Les fichiers peuvent inclure :
   - Plans architecturaux
   - Documents techniques
   - Devis et factures
   - Autres documents partagés

## Notifications

Vous recevrez des notifications lorsque :
- De nouveaux produits sont ajoutés au projet
- L'équipe vous envoie un message
- Des fichiers sont partagés
- Des modifications importantes sont apportées au projet

Configurez vos préférences de notification dans **Compte > Notifications**.
