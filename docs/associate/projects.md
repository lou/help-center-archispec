# Gestion des projets

Les projets sont au cœur d'Archispec. Chaque projet représente un chantier ou une mission architecturale complète.

## Vue d'ensemble d'un projet

Une fois dans un projet, vous avez accès à plusieurs sections :

### Produits
La liste de tous les produits ajoutés au projet. Vous pouvez :
- Ajouter des produits depuis votre bibliothèque
- Créer de nouveaux produits directement dans le projet
- Modifier les quantités et prix
- Archiver des produits
- Dupliquer des produits

### Zones
Organisez votre projet en zones (pièces, étages, etc.) pour mieux structurer vos produits.

### Chat
Communiquez avec votre équipe et vos clients directement dans le projet.

### Fichiers
Gérez tous les fichiers liés au projet (plans, documents, etc.).

### Paramètres
Configurez les paramètres du projet :
- Informations générales
- Membres et permissions
- Client et invitations
- Tarification
- Partage

## Ajouter des produits à un projet

### Depuis la bibliothèque

1. Dans votre projet, cliquez sur **Produits**
2. Cliquez sur **Ajouter depuis la bibliothèque**
3. Sélectionnez les produits que vous souhaitez ajouter
4. Cliquez sur **Ajouter au projet**

### Créer un nouveau produit

1. Dans la section **Produits** du projet, cliquez sur **Nouveau produit**
2. Remplissez les informations du produit :
   - Nom
   - Référence
   - Type de produit
   - Fabricant
   - Fournisseur
   - Prix unitaire
   - Quantité
   - Zone (optionnel)
3. Ajoutez des images si nécessaire
4. Cliquez sur **Créer**

### Depuis l'extension Chrome

L'extension Chrome permet d'ajouter rapidement des produits depuis n'importe quel site web. Consultez la [documentation de l'extension](../extension/installation) pour plus de détails.

## Gérer les zones

Les zones permettent d'organiser votre projet par pièces, étages ou toute autre division logique :

1. Accédez aux **Paramètres** du projet
2. Cliquez sur **Zones**
3. Cliquez sur **Ajouter une zone**
4. Donnez un nom à la zone
5. Réorganisez les zones par glisser-déposer si nécessaire

## Partager un projet avec un client

1. Accédez aux **Paramètres** du projet
2. Cliquez sur **Client**
3. Si le client n'existe pas encore, créez-le
4. Cliquez sur **Inviter le client**
5. Le client recevra une invitation par email

Une fois invité, le client pourra accéder au projet depuis son espace client.

## Exporter un projet

Archispec permet d'exporter vos projets sous différents formats :

- **Excel** : Tableur avec tous les produits et leurs informations
- **PDF** : Devis ou facture formaté
- **Cahier des charges** : Document PDF avec images et descriptions

Pour exporter :
1. Dans la section **Produits** du projet
2. Cliquez sur **Exporter**
3. Choisissez le format souhaité
4. Téléchargez le fichier généré
