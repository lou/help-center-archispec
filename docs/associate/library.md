# Bibliothèque de produits

La bibliothèque de produits est votre catalogue centralisé de tous les produits et matériaux que vous utilisez dans vos projets.

## Accéder à la bibliothèque

1. Dans le menu principal, cliquez sur **Bibliothèque**
2. Sélectionnez **Produits**

Vous verrez tous les produits que vous avez créés ou ajoutés.

## Créer un produit dans la bibliothèque

1. Dans la bibliothèque, cliquez sur **Nouveau produit**
2. Remplissez les informations :
   - **Nom** : Nom du produit
   - **Référence** : Référence unique du produit
   - **Type** : Catégorie du produit (carrelage, peinture, etc.)
   - **Fabricant** : Nom du fabricant
   - **Fournisseur** : Nom du fournisseur
   - **Prix unitaire** : Prix de base du produit
   - **Description** : Description détaillée (optionnel)
3. Ajoutez des images en glissant-déposant ou en cliquant sur **Ajouter des images**
4. Cliquez sur **Créer**

Le produit est maintenant disponible dans votre bibliothèque et peut être ajouté à n'importe quel projet.

## Modifier un produit

1. Dans la bibliothèque, cliquez sur le produit que vous souhaitez modifier
2. Cliquez sur **Modifier**
3. Modifiez les informations souhaitées
4. Cliquez sur **Enregistrer**

**Note** : Les modifications apportées à un produit dans la bibliothèque n'affectent pas les produits déjà ajoutés aux projets. Pour modifier un produit dans un projet spécifique, modifiez-le directement depuis le projet.

## Rechercher des produits

Utilisez la barre de recherche en haut de la bibliothèque pour trouver rapidement un produit par :
- Nom
- Référence
- Fabricant
- Type

## Organiser votre bibliothèque

Vous pouvez filtrer les produits par :
- **Type** : Filtrer par catégorie de produit
- **Fabricant** : Filtrer par fabricant
- **Fournisseur** : Filtrer par fournisseur

## Ajouter un produit de la bibliothèque à un projet

1. Dans la bibliothèque, sélectionnez un ou plusieurs produits
2. Cliquez sur **Ajouter à un projet**
3. Sélectionnez le projet de destination
4. Cliquez sur **Ajouter**

Vous pouvez également ajouter des produits depuis la page du projet en utilisant l'option **Ajouter depuis la bibliothèque**.
