# Filtrer les produits

Utilisez les filtres pour identifier rapidement les produits pertinents et constituer vos sélections.

## Accéder aux filtres

1. Accédez à la liste des produits de votre projet
2. La barre de filtres se trouve à gauche de la liste
3. Cliquez sur un filtre pour l'activer

## Types de filtres disponibles

### Filtre par statut
Filtrez par statut du produit

### Filtre par zone
Filtrez les produits par zone (pièce, étage, etc.) :

1. Cliquez sur le filtre **Zone**
2. Sélectionnez une ou plusieurs zones
3. La liste se met à jour automatiquement

### Filtre par type de produit
Filtrez par catégorie de produit :

1. Cliquez sur le filtre **Type**
2. Sélectionnez un ou plusieurs types de produits
3. La liste se met à jour

### Recherche textuelle
Utilisez la barre de recherche pour trouver des produits par nom du produit.
Tapez simplement votre recherche dans le champ "Rechercher par titre".

### Filtrer les produits archivés
Activez le switch « Produits archivés » pour afficher ou masquer les produits archivés.
Lorsque le filtre est activé, seuls les produits archivés sont affichés.
Lorsqu'il est désactivé, les produits archivés sont exclus de la liste.

## Combiner plusieurs filtres

Vous pouvez activer plusieurs filtres simultanément pour affiner votre recherche :

1. Activez un premier filtre
2. Activez un deuxième filtre
3. Les deux filtres sont combinés avec un ET logique
4. Seuls les produits correspondant à tous les filtres actifs s'affichent

## Réinitialiser les filtres

Pour effacer tous les filtres actifs, cliquez sur **Réinitialiser** en haut à droite de la barre de filtres.
