# Sélection

Sélectionnez un ou plusieurs produits pour effectuer des actions en lot.

## Sélectionner des produits

### Sélectionner un produit
1. Cliquez sur la case à cocher à gauche du produit
2. Le produit est sélectionné (marqué d'une coche)

### Sélectionner plusieurs produits
1. Cliquez sur les cases à cocher de chaque produit souhaité
2. Tous les produits sélectionnés sont mis en surbrillance

### Sélectionner tous les produits visibles
1. Cliquez sur la case à cocher "Tout sélectionner" dans la barre d'actions située au dessus de la liste
2. Tous les produits actuellement visibles sont sélectionnés

**Note** : Seuls les produits visibles (après filtres) sont sélectionnés, pas tous les produits du projet.

## Indicateurs visuels
Les produits sélectionnés sont mis en surbrillance avec :
- Une coche dans la case à cocher
- Un fond légèrement coloré
- Un compteur dans la barre d'actions indiquant le nombre de produits sélectionnés


## Actions sur les produits sélectionnés

Une fois des produits sélectionnés, des boutons apparaissent dans la barre d'actions.


### Actions
Le bouton "Actions" permet d'effectuer des actions de groupe sur les produits selectionnés.

#### Modifier en lot
1. Sélectionnez plusieurs produits
2. Cliquez sur **Actions** > **Mettre à jour**
3. Choisissez les champs à modifier :
   - Statut
   - Zone
4. Entrez les nouvelles valeurs
5. Cliquez sur **Sauvegarder**

#### Archiver
1. Sélectionnez les produits à archiver
2. Cliquez sur **Archiver**
3. Confirmez l'action


#### Dupliquer
1. Sélectionnez les produits à dupliquer
2. Cliquez sur **Dupliquer**
3. Choisissez la zone dans laquelle vous souhaitez dupliquer les produits
4. Cliquez sur **Dupliquer**

### Exporter
Le Bouton "Exporter" permet d'exporter les produits dans différents formats.

#### Exporter en PDF
1. Sélectionnez les produits à exporter
2. Cliquez sur **Exporter** > **Carnet de détails**
3. Choisissez les champs à exporter
4. Cliquez sur **Exporter**
5. Le fichier en format PDF est généré
6. Vous pourrez télécharger le fichier dans la section "Fichiers" de votre projet

#### Exporter en feuille de calcul
1. Sélectionnez les produits à exporter
2. Cliquez sur **Exporter** > **Feuille de calcul**
3. Choisissez les champs à exporter
4. Cliquez sur **Exporter**
5. Le fichier en format Excel est généré
6. Vous pourrez télécharger le fichier dans la section "Fichiers" de votre projet
