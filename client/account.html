Mon compte

Gérez vos informations personnelles et vos préférences depuis votre compte.

Informations personnelles​

  1. Accédez à Compte dans le menu
  2. Cliquez sur Modifier
  3. Mettez à jour :
    • Votre nom
    • Votre prénom
    • Votre email
    • Votre téléphone
  4. Cliquez sur Enregistrer

Notifications​

Configurez comment vous souhaitez recevoir les notifications :

  1. Accédez à Compte > Notifications
  2. Activez ou désactivez les types de notifications :
    • Nouveaux produits : Recevoir une notification quand un produit est ajouté
    • Messages : Recevoir une notification pour les nouveaux messages
    • Fichiers : Recevoir une notification quand un fichier est partagé
    • Modifications de projet : Recevoir une notification pour les changements importants

Vous pouvez choisir de recevoir les notifications :

  • Par email
  • Dans l'application (badge de notification)

Sécurité​

Pour votre sécurité, il est recommandé de :

  • Utiliser un mot de passe fort
  • Ne jamais partager vos identifiants
  • Vous déconnecter si vous utilisez un ordinateur partagé

Déconnexion​

Pour vous déconnecter :

  1. Cliquez sur votre nom en haut à droite
  2. Sélectionnez Déconnexion
