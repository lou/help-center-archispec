Utiliser l'extension Chrome

L'extension Archispec vous permet d'ajouter rapidement des produits depuis n'importe quel site web à vos projets.

Ajouter un produit depuis un site web​

Méthode 1 : Depuis la page du produit​

  1. Naviguez vers la page d'un produit sur n'importe quel site web
  2. Cliquez sur l'icône de l'extension Archispec dans la barre d'outils
  3. L'extension détecte automatiquement les informations du produit :
    • Nom
    • Prix
    • Images
    • Description
  4. Vérifiez et complétez les informations si nécessaire
  5. Sélectionnez le projet de destination
  6. Cliquez sur Ajouter au projet

Méthode 2 : Sélection manuelle​

Si l'extension ne détecte pas automatiquement le produit :

  1. Sur la page web, sélectionnez le texte contenant le nom du produit
  2. Cliquez sur l'icône de l'extension
  3. L'extension utilisera le texte sélectionné comme nom du produit
  4. Complétez les autres informations manuellement
  5. Ajoutez des images en glissant-déposant ou en copiant l'URL
  6. Sélectionnez le projet
  7. Cliquez sur Ajouter au projet

Modifier les informations avant d'ajouter​

Avant d'ajouter un produit, vous pouvez modifier :

  • Nom : Modifiez le nom du produit
  • Référence : Ajoutez une référence unique
  • Prix : Corrigez ou ajoutez le prix
  • Description : Complétez ou modifiez la description
  • Images : Ajoutez ou supprimez des images
  • Type de produit : Sélectionnez le type depuis votre bibliothèque
  • Fabricant : Ajoutez le nom du fabricant
  • Fournisseur : Ajoutez le nom du fournisseur

Sélectionner le projet de destination​

  1. Dans l'interface de l'extension, cliquez sur Sélectionner un projet
  2. Choisissez le projet dans la liste
  3. Vous pouvez également créer un nouveau projet directement depuis l'extension

Ajouter à la bibliothèque​

Au lieu d'ajouter directement à un projet, vous pouvez ajouter le produit à votre bibliothèque :

  1. Dans l'interface de l'extension, cochez Ajouter à la bibliothèque
  2. Le produit sera disponible dans votre bibliothèque
  3. Vous pourrez l'ajouter à n'importe quel projet plus tard

Conseils d'utilisation​

Meilleures pratiques​

  • Vérifiez les informations : L'extension détecte automatiquement les informations, mais vérifiez toujours avant d'ajouter
  • Complétez les détails : Ajoutez la référence, le type et d'autres informations pour faciliter la gestion
  • Utilisez la bibliothèque : Pour les produits récurrents, ajoutez-les d'abord à la bibliothèque
  • Organisez par projet : Sélectionnez toujours le bon projet pour éviter les erreurs

Sites compatibles​

L'extension fonctionne sur la plupart des sites web de vente de produits. Elle détecte automatiquement :

  • Les sites e-commerce standards
  • Les catalogues en ligne
  • Les sites de fabricants
  • Les pages de produits individuelles

Résolution de problèmes​

Si l'extension ne détecte pas correctement un produit :

  1. Essayez de sélectionner manuellement le texte du nom du produit
  2. Vérifiez que vous êtes bien connecté à Archispec
  3. Actualisez la page et réessayez
  4. Complétez les informations manuellement si nécessaire
