Configuration de l'extension

Personnalisez le comportement de l'extension selon vos besoins.

Paramètres de l'extension​

Pour accéder aux paramètres :

  1. Cliquez sur l'icône de l'extension
  2. Cliquez sur l'icône de paramètres (⚙️) ou sur Paramètres

Options disponibles​

Projet par défaut​

Définissez un projet par défaut pour que l'extension propose automatiquement ce projet lors de l'ajout de produits.

  1. Dans les paramètres, sélectionnez Projet par défaut
  2. Choisissez un projet dans la liste
  3. L'extension utilisera ce projet par défaut (vous pourrez toujours le changer)

Ajout automatique à la bibliothèque​

Activez cette option pour que tous les produits ajoutés soient automatiquement enregistrés dans votre bibliothèque.

  1. Activez Ajouter automatiquement à la bibliothèque
  2. Tous les nouveaux produits seront disponibles dans votre bibliothèque

Notifications​

Configurez les notifications de l'extension :

  • Notifications de succès : Recevoir une notification quand un produit est ajouté avec succès
  • Notifications d'erreur : Recevoir une notification en cas d'erreur

Raccourcis clavier​

L'extension peut être ouverte rapidement avec un raccourci clavier :

  1. Allez dans chrome://extensions/shortcuts
  2. Trouvez l'extension Archispec
  3. Définissez un raccourci personnalisé pour ouvrir l'extension

Synchronisation​

L'extension se synchronise automatiquement avec votre compte Archispec :

  • Vos projets sont mis à jour en temps réel
  • Les modifications sont synchronisées immédiatement
  • Vous pouvez utiliser l'extension sur plusieurs ordinateurs avec le même compte

Mise à jour​

L'extension se met à jour automatiquement depuis le Chrome Web Store. Vous recevrez une notification lorsqu'une nouvelle version est disponible.

Pour vérifier manuellement les mises à jour :

  1. Allez dans chrome://extensions
  2. Activez Mode développeur (en haut à droite)
  3. Cliquez sur Mettre à jour pour toutes les extensions

Support​

Si vous rencontrez des problèmes avec l'extension :

  1. Vérifiez que vous êtes connecté à Archispec
  2. Vérifiez que l'extension est à jour
  3. Réinstallez l'extension si nécessaire
  4. Contactez le support Archispec si le problème persiste
