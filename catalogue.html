Catalogue

Le catalogue est votre bibliothèque centrale de produits. Gérez tous vos produits au même endroit et réutilisez-les dans vos projets.

Accéder au catalogue​

  1. Dans le menu principal, cliquez sur Catalogue ou Bibliothèque
  2. Tous vos produits sont affichés

Ajouter un produit au catalogue​

Créer un nouveau produit​

  1. Cliquez sur Nouveau produit
  2. Remplissez les informations du produit :

Informations de base​

  • Nom : Nom du produit (obligatoire)
  • Référence : Référence interne ou du fabricant
  • Type de produit : Catégorie du produit
  • Description : Description détaillée

Informations commerciales​

  • Fabricant : Nom du fabricant
  • Fournisseur : Nom du fournisseur
  • Prix unitaire : Prix de référence
  • Lien web : URL du produit sur le site du fournisseur

Images​

  • Ajoutez des images du produit
  • La première image sera l'image principale
  1. Cliquez sur Créer

Importer depuis un projet​

Ajoutez un produit du catalogue depuis un projet :

  1. Dans le projet, accédez à la liste des produits
  2. Cliquez sur le produit
  3. Cliquez sur Ajouter au catalogue
  4. Le produit est copié dans le catalogue

Importer depuis l'extension Chrome​

L'extension Chrome permet d'ajouter rapidement des produits au catalogue :

  1. Utilisez l'extension pour extraire un produit depuis un site web
  2. Sélectionnez Ajouter au catalogue au lieu d'un projet spécifique
  3. Le produit est ajouté au catalogue

Organiser le catalogue​

Créer des catégories​

Organisez vos produits en catégories :

  1. Cliquez sur Catégories dans le menu latéral
  2. Cliquez sur Nouvelle catégorie
  3. Entrez le nom de la catégorie
  4. Cliquez sur Créer

Assigner un produit à une catégorie​

  1. Cliquez sur le produit
  2. Dans les détails, sélectionnez la Catégorie
  3. Cliquez sur Enregistrer

Créer des collections​

Les collections permettent de regrouper des produits pour un usage spécifique :

  1. Cliquez sur Collections dans le menu latéral
  2. Cliquez sur Nouvelle collection
  3. Donnez un nom à la collection
  4. Ajoutez des produits à la collection
  5. Cliquez sur Créer

Rechercher dans le catalogue​

Recherche textuelle​

Utilisez la barre de recherche pour trouver des produits par :

  • Nom
  • Référence
  • Fabricant
  • Fournisseur
  • Description

Filtres​

Filtrez les produits par :

  • Catégorie : Type de produit
  • Fabricant : Marque
  • Fournisseur : Fournisseur
  • Prix : Plage de prix
  • Date d'ajout : Quand le produit a été ajouté

Modifier un produit​

  1. Cliquez sur le produit dans le catalogue
  2. Modifiez les informations souhaitées
  3. Cliquez sur Enregistrer

Note : Les modifications dans le catalogue n'affectent pas automatiquement les produits déjà ajoutés aux projets. Vous pouvez choisir de mettre à jour les projets ou de garder les anciennes versions.

Dupliquer un produit​

  1. Cliquez sur le menu (⋮) à côté du produit
  2. Sélectionnez Dupliquer
  3. Le produit est copié avec le suffixe " (copie)"
  4. Modifiez la copie selon vos besoins

Supprimer un produit​

  1. Cliquez sur le menu (⋮) à côté du produit
  2. Sélectionnez Supprimer
  3. Confirmez l'action

Attention : La suppression d'un produit du catalogue ne le supprime pas des projets où il est utilisé.

Ajouter un produit du catalogue à un projet​

  1. Ouvrez le projet
  2. Accédez à la section Produits
  3. Cliquez sur Ajouter depuis la bibliothèque
  4. Recherchez et sélectionnez les produits du catalogue
  5. Cliquez sur Ajouter au projet

Les produits sont ajoutés avec toutes leurs informations (prix, images, descriptions).

Synchronisation des prix​

Si vous modifiez le prix d'un produit dans le catalogue, vous pouvez choisir de mettre à jour les projets :

  1. Après avoir modifié le prix, une notification apparaît
  2. Cliquez sur Mettre à jour les projets
  3. Sélectionnez les projets à mettre à jour
  4. Cliquez sur Appliquer

Exporter le catalogue​

Exportez votre catalogue complet :

  1. Cliquez sur Exporter dans le menu
  2. Choisissez le format :
    • Excel : Tableur avec tous les produits
    • CSV : Format CSV pour import dans d'autres systèmes
  3. Le fichier est généré et téléchargé

Importer un catalogue​

Importez plusieurs produits à la fois :

  1. Cliquez sur Importer dans le menu
  2. Sélectionnez le fichier (Excel ou CSV)
  3. Mappez les colonnes aux champs de produits
  4. Vérifiez l'aperçu
  5. Cliquez sur Importer

Les produits sont ajoutés au catalogue.
