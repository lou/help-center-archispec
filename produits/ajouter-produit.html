Ajouter un produit

Ajoutez des produits à votre projet depuis différentes sources.

Méthodes d'ajout​

Depuis la bibliothèque​

Ajoutez des produits déjà présents dans votre bibliothèque :

  1. Dans votre projet, accédez à la section Produits
  2. Cliquez sur Ajouter depuis la bibliothèque
  3. Utilisez les filtres pour trouver les produits souhaités
  4. Cochez les produits à ajouter
  5. Cliquez sur Ajouter au projet

Les produits sont ajoutés avec leurs informations complètes (prix, images, descriptions, etc.).

Créer un nouveau produit​

Créez un produit directement dans le projet :

  1. Dans la section Produits du projet, cliquez sur Nouveau produit
  2. Remplissez les informations du produit :

Informations de base​

  • Nom : Nom du produit (obligatoire)
  • Référence : Référence interne ou du fabricant
  • Type de produit : Catégorie du produit
  • Description : Description détaillée

Informations commerciales​

  • Fabricant : Nom du fabricant
  • Fournisseur : Nom du fournisseur
  • Prix unitaire : Prix pour une unité
  • Quantité : Quantité dans le projet

Organisation​

  • Zone : Zone du projet (pièce, étage, etc.)
  • Statut : Actif, Archivé, Sélectionné

Visibilité​

  • Visible pour le client : Cochez si le produit doit être visible par le client

  1. Ajoutez des images si nécessaire :

    • Cliquez sur Ajouter des images
    • Sélectionnez les fichiers depuis votre ordinateur
    • Glissez-déposez les images ou cliquez pour parcourir

  2. Cliquez sur Créer

Depuis l'extension Chrome​

L'extension Chrome permet d'ajouter rapidement des produits depuis n'importe quel site web :

  1. Installez l'extension Chrome (voir Installation de l'extension)
  2. Naviguez sur un site web contenant un produit
  3. Cliquez sur l'icône de l'extension
  4. L'extension extrait automatiquement les informations du produit
  5. Vérifiez et complétez les informations si nécessaire
  6. Sélectionnez le projet de destination
  7. Cliquez sur Ajouter au projet

Consultez la documentation de l'extension pour plus de détails.

Modifier un produit​

Après avoir ajouté un produit, vous pouvez le modifier :

  1. Cliquez sur le produit dans la liste
  2. Modifiez les informations souhaitées
  3. Cliquez sur Enregistrer

Dupliquer un produit​

Pour créer une copie d'un produit existant :

  1. Cliquez sur le menu (⋮) à côté du produit
  2. Sélectionnez Dupliquer
  3. Le produit est copié avec le suffixe " (copie)"
  4. Modifiez la copie selon vos besoins

Importer des produits​

Vous pouvez importer plusieurs produits à la fois depuis un fichier Excel ou CSV :

  1. Dans la section Produits, cliquez sur Importer
  2. Sélectionnez le fichier à importer
  3. Mappez les colonnes du fichier aux champs de produits
  4. Vérifiez l'aperçu des produits à importer
  5. Cliquez sur Importer

Les produits sont ajoutés au projet avec les informations du fichier.

Conseils​

  • Utilisez la bibliothèque pour les produits récurrents
  • Créez des produits dans le projet pour les produits spécifiques à un projet
  • Utilisez l'extension Chrome pour extraire rapidement des produits depuis des sites web
  • Ajoutez toujours des images pour faciliter l'identification des produits
