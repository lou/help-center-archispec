Mode prix

Le mode prix est une fonctionnalité optionnelle qui permet d'afficher une vue détaillée des prix dans la liste des produits.

Activer le mode prix​

  1. Accédez à la liste des produits
  2. Utilisez le bouton ou le toggle Mode prix dans la barre d'actions située au dessus de la liste
  3. Le mode prix s'active et affiche des informations de prix supplémentaires

Fonctionnement​

Le mode Prix vous permet de visualiser et d'analyser en détail les coûts de votre projet.

Affichage des prix dans la liste​

Lorsque le mode Prix est activé :

  • le prix de chaque produit est affiché directement dans la liste
  • le prix total de tous les produits est visible dans la barre d'actions située au-dessus de la liste
  • le prix total de chaque zone est affiché à droite du nom de la zone correspondante

Cela vous permet d'avoir une vision immédiate des coûts, à la fois au niveau global et par zone.

Détail et répartition des prix​

En cliquant sur le prix total du projet dans la barre d'actions, vous accédez à un écran de découpage des prix. Cette vue vous permet d'analyser les montants selon :

  • les filtres appliqués
  • la sélection de produits en cours
  • le projet global

Pour chaque répartition, vous pouvez consulter :

  • la marge
  • le montant hors taxes (HT),
  • le montant toutes taxes comprises (TTC)

Cette fonctionnalité est particulièrement utile pour affiner vos choix, comparer différents scénarios et piloter la rentabilité de votre projet.

Désactiver le mode prix​

Pour revenir à la vue normale :

  1. Cliquez à nouveau sur le toggle Mode prix dans la barre d'actions située au dessus de la liste
  2. Le mode prix se désactive et vous revenez à la vue standard

Préférences utilisateur​

Le mode prix est mémorisé. Si activé, il se réactivera automatiquement lors de vos prochaines visites.
