Fichiers

Gérez tous les fichiers liés à votre projet : plans, documents, images et plus encore.

Accéder aux fichiers​

  1. Ouvrez votre projet
  2. Cliquez sur Fichiers dans le menu latéral du projet
  3. Tous les fichiers du projet s'affichent

Ajouter des fichiers​

Glisser-déposer​

  1. Accédez à la section Fichiers
  2. Glissez les fichiers depuis votre ordinateur vers la zone de dépôt
  3. Les fichiers sont automatiquement téléversés

Parcourir les fichiers​

  1. Cliquez sur Ajouter des fichiers
  2. Sélectionnez les fichiers depuis votre ordinateur
  3. Cliquez sur Ouvrir
  4. Les fichiers sont téléversés

Types de fichiers supportés​

  • Images : JPG, PNG, GIF, WebP
  • Documents : PDF, DOC, DOCX, XLS, XLSX
  • Plans : DWG, DXF (si supporté)
  • Autres : Tous les types de fichiers peuvent être téléversés

Organiser les fichiers​

Créer des dossiers​

  1. Cliquez sur Nouveau dossier
  2. Entrez le nom du dossier
  3. Cliquez sur Créer
  4. Glissez-déposez les fichiers dans le dossier

Renommer un fichier​

  1. Cliquez sur le menu (⋮) à côté du fichier
  2. Sélectionnez Renommer
  3. Entrez le nouveau nom
  4. Cliquez sur Enregistrer

Déplacer un fichier​

  1. Cliquez et maintenez sur le fichier
  2. Glissez-le vers le dossier de destination
  3. Relâchez pour déposer

Partager des fichiers avec le client​

Partager un fichier​

  1. Cliquez sur le menu (⋮) à côté du fichier
  2. Sélectionnez Partager avec le client
  3. Le fichier devient accessible au client dans son interface

Partager un dossier​

  1. Cliquez sur le menu (⋮) à côté du dossier
  2. Sélectionnez Partager avec le client
  3. Tous les fichiers du dossier deviennent accessibles au client

Voir les fichiers partagés​

Pour voir uniquement les fichiers partagés avec le client :

  1. Utilisez le filtre Partagés dans la barre de filtres
  2. Seuls les fichiers partagés s'affichent

Télécharger des fichiers​

Télécharger un fichier​

  1. Cliquez sur le fichier
  2. Le fichier s'ouvre dans un nouvel onglet
  3. Utilisez le menu de votre navigateur pour télécharger

Ou :

  1. Cliquez sur le menu (⋮) à côté du fichier
  2. Sélectionnez Télécharger

Télécharger plusieurs fichiers​

  1. Sélectionnez les fichiers à télécharger (cases à cocher)
  2. Cliquez sur Télécharger dans la barre d'actions
  3. Les fichiers sont téléchargés dans une archive ZIP

Supprimer des fichiers​

  1. Cliquez sur le menu (⋮) à côté du fichier
  2. Sélectionnez Supprimer
  3. Confirmez l'action

Attention : La suppression est définitive et irréversible.

Prévisualiser des fichiers​

Images​

Les images peuvent être prévisualisées directement dans l'interface :

  1. Cliquez sur l'image
  2. Elle s'ouvre en mode plein écran
  3. Utilisez les flèches pour naviguer entre les images

PDF​

Les fichiers PDF peuvent être visualisés dans le navigateur :

  1. Cliquez sur le fichier PDF
  2. Il s'ouvre dans un nouvel onglet

Autres formats​

Pour les autres formats, utilisez le téléchargement pour les ouvrir avec votre application préférée.

Rechercher des fichiers​

Utilisez la barre de recherche pour trouver des fichiers par :

  • Nom du fichier
  • Type de fichier
  • Date de modification

Tapez simplement votre recherche dans la barre de recherche en haut de la liste.

Trier les fichiers​

Vous pouvez trier les fichiers par :

  • Nom : Ordre alphabétique
  • Date : Date de modification (plus récent en premier ou plus ancien)
  • Taille : Taille du fichier
  • Type : Type de fichier

Cliquez sur l'en-tête de colonne pour changer le tri.
