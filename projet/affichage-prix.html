Affichage des prix

Configurez comment les prix sont affichés dans votre projet.

Afficher les prix avec ou sans taxes​

  1. Accédez aux Paramètres du projet
  2. Cliquez sur Affichage des prix dans le menu latéral
  3. Choisissez l'option d'affichage :
    • Avec taxes : Les prix affichés incluent les taxes (TTC)
    • Sans taxes : Les prix affichés sont hors taxes (HT)
  4. Le changement est enregistré automatiquement

Cette option détermine comment les prix sont affichés dans toute l'interface du projet mais aussi dans l'interface client.
