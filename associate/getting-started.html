Premiers pas avec l'application Archispec

L'application web Archispec est l'outil principal pour gérer vos projets architecturaux, vos produits et collaborer avec vos clients.

Connexion​

  1. Accédez à l'application Archispec
  2. Cliquez sur Se connecter ou S'inscrire si vous n'avez pas encore de compte
  3. Entrez vos identifiants

Créer votre première entreprise​

Après votre inscription, vous devrez créer ou rejoindre une entreprise :

  1. Cliquez sur Créer une entreprise ou Rejoindre une entreprise
  2. Remplissez les informations de l'entreprise (nom, adresse, etc.)
  3. Configurez les paramètres de base

Navigation principale​

L'application se compose de plusieurs sections principales :

  • Projets : Gérez tous vos projets architecturaux
  • Bibliothèque : Accédez à votre bibliothèque de produits
  • Entreprises : Gérez les entreprises et leurs membres
  • Compte : Paramètres personnels et préférences

Créer votre premier projet​

  1. Accédez à la section Projets
  2. Cliquez sur Nouveau projet
  3. Remplissez les informations du projet :
    • Nom du projet
    • Description (optionnel)
    • Client (vous pouvez créer un nouveau client ou en sélectionner un existant)
  4. Cliquez sur Créer

Votre projet est maintenant créé et prêt à être utilisé !
